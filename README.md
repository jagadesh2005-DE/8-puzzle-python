# 8 Puzzle and Monkey Banana Problem (AI)

This repository contains Python implementations of two classic Artificial Intelligence problems.

## Problems Implemented

### 1. Eight Puzzle Problem

The Eight Puzzle is a sliding puzzle consisting of a 3×3 grid with eight numbered tiles and one empty space.
The goal is to move the tiles until the puzzle reaches the goal state.

The program demonstrates how AI search techniques can be used to solve the puzzle.

File:

* `eight_tile_problem.py`

### 2. Monkey Banana Problem

The Monkey Banana Problem is a classic AI problem used to illustrate problem-solving using states and actions.

In this problem:

* A monkey wants to reach a banana hanging from the ceiling.
* A box is available in the room.
* The monkey must move the box, climb on it, and grab the banana.

File:

* `monkey_banana.py`
